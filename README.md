# S4R-TransMamba

## 📋 Overview

S4R-TransMamba is a project developed for SEM_3 EDI (Electronic Design Innovation). This repository focuses on protein structure prediction and analysis, utilizing advanced computational tools for molecular modeling.

## 🗂️ Repository Structure

```
S4R-TransMamba/
├── README.md
├── boltz                          # Boltz protein structure prediction tool
├── boltz_out/                     # Output directory for Boltz predictions
└── boltz_results_multimer/        # Results for multimer structure predictions
```

## 🧬 Key Components

### Boltz Integration
This project integrates **Boltz**, a state-of-the-art protein structure prediction tool that enables:
- Single protein structure prediction
- Multimer complex modeling
- High-accuracy structural analysis

### Output Directories
- **boltz_out/**: Contains raw output files from Boltz predictions
- **boltz_results_multimer/**: Specialized directory for protein complex (multimer) prediction results
